# Hoobank: Next-Generation Payment Method

Hoobank is a FinTech solution designed to simplify identifying the most suitable credit cards for users. By leveraging expert methodologies, Hoobank analyzes key financial metrics such as annual percentage rates (APRs) and annual fees, providing personalized recommendations to help users make informed decisions.

## Technologies Used

- **Frontend**: React, Tailwind CSS
- **Version Control**: Git, GitHub
